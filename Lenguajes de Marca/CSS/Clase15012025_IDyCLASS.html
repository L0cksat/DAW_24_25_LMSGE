Definición de ID

En HTML, tenemos un atributo que se llama "id" y que se referencia como IDentificador.
En un html se puede utilizar un id para identificar un único elemento o etiqueta
dentro de un doucmento con el fin de aplicarle una serie de caracteristicas concretas.
Es decir, No debería haber otro delemento con el mismo nombre de identificaardor (id)
del documento HTML.
Cuando vamos a utilizar un id en HTML, utilizaremos la siguiente estructura:
nombre etiqueta (por ejemplo, img) id = "nombre_identificador"
Ejemplo: img id = "foto" src=...
Cuando vayamos a utilziar un id en CSS, utiolizaremos la siguiente esturctura:
#nombre_etiqueta: proipiedades a aplicar
Ejemplo: #foto{ width = "400px"}

Definición de CLASS

El valor del atributo class, a diferencia del valor del atributo id, puede ser utilizado
en más de un elemento de nuestro documento HTML.
Esto es muy benificioso cuando tengamos que aplicar los mismos estilos a diferentes elementos
dado que nos permitirá reducir las lineas de codigo en nuestro archivo CSS.
Cuando vayamos aplicar una clase en HTML, seguiremos l siguiente estructura:
nombre_etiqueta class = "nombre_clase" --> img class="foto" src=...
Cuando vayamos a aplicar una clase en CSS, seguiremos la siguiente estructura:
.foto{width = "400px"}
