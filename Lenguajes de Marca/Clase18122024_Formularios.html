Formularios HTML

Los formularios en HTML, son elementos que nos van a dotar de cierta funcionalidad y dinamismo a nuestras páginas web. Nos van a permitir interactuar con nuestras páginas web y por lo tanto, son elementos fundamentales para poder
recoger información para posteriormente gestionarla.
Los formularios nos van a permitir dos tipos de funcionalidades principales:
  • Recoger información del usuario
  • Enviar esos datos que se han guardado, para posteriormente procesarlos o para poder realizar
    cualquier tipo de operación que se requiera con ellos.

Los principales objetivos que tenemos que tener en cuenta a la hora de diseñar un formulario HTML,
son los siguientes:
  • El formularo tiene que ser lo más sencillo posible a la hora de insertar datos por parte del usuario.
  • La experiencia del usuario/usabilidad (UX) tiene que ser lo más agradable posible.
  • Los datos introducidos en los formularios, tienen que tener un formato predecible y esperado.
    P.E: Si vamos a introducir el nombre de usuario, facilitemos al usuario que sepa dónde lo tiene
    que introducir facilitandole el apartado nombre para que lo vea y sepa que es la zona correcta...
  • Reducir los errores al introducir datos en la medida de lo posible y comunicar claramente al usuario
    si está dando algún tipo de error al introducirlos...

Etiquetas básicas para la creación de un formulario en HTML

Para crear un formulariuo en HTML, utilizaremos la etiqueta form con apertura y cierre.
A parte de utilizar la etiqueta form, que establece que se va a usar/crear un formulario, tenemos que tener
en cuenta una serie de propiedades que vamos a usar con ella:
  • action:Indica la URL o BBDD, web, correo...(Dirección Backend) a la que queremos enviar
    la información de nuestro formulario.
  • method: Método puede ser get o post. Get se utiliza cuando queremos
    mostrar la información que se envía de nuestro formulario a través del protcolo HTTP/HTTPS de nuestra
    web.
    Post se utiliza para ocultar la información que se envía de nuestro a través del protocolo
    HTTP/HTTPS de nuestra web.
  • name: Se utiliza para dar un nombre al formulario.
  • target: Se utiliza para poder arbir el formulario en una pestaña nueva (target=_blank).
  • entype: Codificación de información mediante token.
  • autocomplete: Tiene una función on u off, que sirve para activar y desactivar el autocmpletado de
    nuestra web.

Tipo de datos que podemos gestionar en un Formulario HTML

En un formulario, podemos gestionar cualquier tipo de información que se quiera/requiera. Pero los más
utilizados o que vamos a utilizar más son los siguientes:
  • Información de texto: Nombres, Apellidos, direcciones físicas, etc. Para esto, utilizaremos
    la etiqueta input y textarea. (input type=text y el textarea con etiqueta de apertura y cierre)
  • Números y Cantidades: edades, precios, cuantías, etc. Utilizaremos el elemento input pero del Tipo number (input type=number)
  • Fechas y horas: Fechas de nacimientos, calendarios, ect. Ultilizaremos el input date, input week, etc.
  • Opciones de verdadero o falso: Utilizaremos el tipo input type=checkbox
  • Opciones únicas: podremos elegir entre 2 o más opciones. input type=radio o la etiqueta select
  • Opciones únicas simples: Donde utilizaremos el select multiple, input checkbox, etc.
  • Opciones de listas abiertas: datalist
  • Seleccionar colores: input type=color
  • Seleccionar un archivo: input type=file

Campos de Entrada de Información Númericos

Para poder obtener información númerica de nuestros formularios, podemos utilizar dos tipos de campos existentes:
  • input type=number con el cuál estamos indicando que el valor de entrada será númerico.
  • campo de rango: input type=range
  • Los atrbutos que van asociados a estos: min (establece un valor mínimo para pasar la validación)
    max(número máximo permitido para pasar la validación), el step (permite la variaación entre Números
    para pasar la valdicación).

Campos de Entrada Númericos: fecha y hora

Listas de Selección en Formularios

En los formularios, se utilizan listas de selección, cuando por ejemplo, las casillas de verificación se nos quedan
cortas o necesitamos mostrar una lista más extensa de datos.
A través de estas listas, podemos mostrar al usuario varias opciones disonibles para elegir una de ellas.
Entre los tipos de listados que tenemos para usar en formularios, podemos destacar las siguientes:
  • Listas Cerradas: Son listas de opciones en donde tenemos los elementos select y option
  • Listas Abiertas: Son listas de opciones en donde tenemos el elemento datalist

Agrupación de Campos

Son etiquetas que nos permiten organizar los elementeos de nuestros formularios reagrupándolos por categorías, temáticas
entre otro tipos de cosas.
  • Fieldset: Agrupar de manera visual los campos de entrada por temática
  • Legend: Una leyenda para la etiqueta fieldset
  • Label: Una etiqueta que sirve para relacionar el campo y el texto de un formulario

Botones en Formularios HTML

Podemos distinguir 4 tipos de botones para usarlos en formularios:
  • Boton de Envío de Formulario: input type = submit
  • Boton de Envío con Imágen: input type = image
  • Boton de Reseteo / Borrado: input type = reset
  • Boton sin funcionalidad: input type = button o definido con las etiquetas de apertura y cierra button.
