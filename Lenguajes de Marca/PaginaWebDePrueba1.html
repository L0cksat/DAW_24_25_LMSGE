Formatos de Textos

  • Etiqueta b: sirve para poner texto en negrita
  • Etiqueta i: sirve para poner texto en cursiva
  • Etiqueta del: sirve para poner texto tachado
  • Etiqueta em: similar a cursiva, pero sirve para dar énfasis
  • Etiqueta small: sirve para poner texto en pequeño
  • Etiqueta sup: Es una etiqueta de superríndice Ejemplo: x2
  • Etiqueta sub: Es una etiqueta de subíndice Ejemplo: H2O
  • Etiqueta mark: Es para subrayar un texto
  • Etiqueta q: Sirve para poner citas pequeñas de texto
  • Etiqueta cite: Sirve para poner Títulos de trabajos
  • Etiqueta address: Sirve para poner una dirreción.
